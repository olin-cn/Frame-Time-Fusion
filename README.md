# Frame-Time-Fusion
# Trajectory Reconstruction and Kalman Prediction

## Overview

This project simulates a sine-wave trajectory, generates voltage relaxation data, reconstructs the trajectory, and predicts future points using a Kalman filter.

## Files

```text
Simulated Trajectory Generation.py
Trajectory Reconstruction and Prediction.py
LICENSE
```

## Requirements

```bash
pip install numpy pandas matplotlib
```

## Usage

Run the trajectory data generation script:

```bash
python "Simulated Trajectory Generation.py"
```

Run the reconstruction and prediction script:

```bash
python "Trajectory Reconstruction and Prediction.py"
```

## Outputs

The scripts generate trajectory data, reconstruction results, prediction metrics, and visualization figures.

## Features

- Sine-wave trajectory simulation
- Voltage relaxation signal generation
- Dynamic-programming trajectory reconstruction
- Moving-average smoothing
- Kalman-filter future-point prediction
- Accuracy and speed evaluation
- Figure export
